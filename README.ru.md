# Проверка Хэша

Простая CLI-программа, которая вычисляет и сравнивает (опционально) хэш-суммы.

## Возможности
- Вычисляет хэш файла
- Доступно 4 хэша: SHA256, MD5, SHA1, SHA512
- Сравнивает его с введёным хэшем (опционально, используйте `/c`)

## Требования
- Python 3

## Использование
- Запуск скрипта: `python hash_checker.py`
- Список команд:
    - `/h` – помощь
    - `/t` – сменить тип хэша
    - `/c` – включить/выключить сравнение
    - `/q` – выйти

## Сборка в `.exe`
- Установка зависимостей (используйте venv):
    1. Используя pip: `pip install pyinstaller` 
    2. Либо используя uv: `uv sync --no-dev`
- Сборка: `pyinstaller --onefile hash_checker.py`
- Приложение будет в: dist/hash_checker.exe
